Mastering Classes and Objects in Java

Welcome to our comprehensive guide on classes and objects in Java! In this tutorial, we'll explore the fundamental concepts of object-oriented programming (OOP), focusing on classes and objects.

Understanding Classes

In Java, a class is a blueprint or template for creating objects. It defines the properties (attributes) and behaviors (methods) that objects of that class will have. Classes are the building blocks of object-oriented programming and provide a way to structure and organize code.

Creating Classes

To create a class in Java, you use the class keyword followed by the class name and the body of the class enclosed in curly braces. Let's take a closer look at the syntax:

public class ClassName {

 // Class members (fields, methods, constructors)

}

Defining Attributes and Methods

Attributes are the variables or fields that represent the state of an object, while methods are the functions or behaviors that objects can perform. Here's how you define attributes and methods within a class:

        public class MyClass {
            // Attributes
            private int attribute1;
            private String attribute2;
    
            // Methods
            public void method1() {
                // Method body
            }
    
            public int method2() {
                // Method body
            }
        }

Creating Objects

Once you've defined a class, you can create objects (instances) of that class using the new keyword followed by the class name and parentheses. This process is called instantiation. Let's see how it's done:

        MyClass object1 = new MyClass();
        MyClass object2 = new MyClass();

Accessing Class Members

You can access the attributes and methods of an object using the dot (.) operator. Here's how you can access and modify attributes and call methods:

        object1.attribute1 = 10;
        object1.method1();

Let's practise your understanding about Classes and Objects

Java code that demonstrates the declaration and initialization of Classes and Objects 

        
class Car {
    
    String brand;
    String model;
    int year;
    
   
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
    
   
    public void displayInfo() {
        System.out.println("Brand " + brand);
        System.out.println("Model " + model);
        System.out.println("Year " + year);
    }
}


public class Main {
    public static void main(String[] args) {
      
        Car myCar = new Car("Toyota", "Camry", 2020);
        
        
        myCar.displayInfo();
    }
}
