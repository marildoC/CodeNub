CSS Selectors and Styling

Welcome to the course on CSS Selectors and Styling! In this module, we'll explore the power of CSS selectors and how they're used to style HTML elements.

Understanding CSS Selectors

CSS selectors are patterns used to select and style elements in an HTML document. They allow developers to target specific elements based on their attributes, relationships with other elements, and more. Let's delve into some common CSS selectors:

  • Element Selector: Selects HTML elements based on their tag name. For example, p selects all paragraphs.
  • Class Selector: Selects elements with a specific class attribute. For example, .classname selects all elements with the class "classname".
  • ID Selector: Selects a single element with a specific ID attribute. For example, #idname selects the element with the ID "idname".
  • Attribute Selector: Selects elements based on their attribute values. For example, [attribute=value] selects elements with the specified attribute and value.
  • Descendant Selector: Selects elements that are descendants of another element. For example, parent descendant selects all descendants of the "parent" element.
  • Child Selector: Selects elements that are direct children of another element. For example, parent > child selects all direct children of the "parent" element.
  • Pseudo-class Selector: Selects elements based on their state or position. For example, :hover selects an element when the user hovers over it.
  • Pseudo-element Selector: Selects parts of an element. For example, ::first-line selects the first line of a paragraph.

Applying Styles with CSS

Once selectors have been defined, CSS properties are used to style the selected elements. Here are some common CSS properties and their applications:

  • Color: Sets the color of text and backgrounds.
  • Font: Controls the font family, size, weight, style, and decoration.
  • Padding and Margin: Adjusts the spacing inside and around elements.
  • Border: Defines the border around elements.
  • Width and Height: Sets the width and height of elements.
  • Background: Specifies background images, colors, and positions.
  • Positioning: Positions elements relative to their normal position.
  • Display: Specifies how elements are displayed.
  • Flexbox and Grid: Modern layout techniques for arranging elements.
  • Transitions and Animations: Adds motion and interactivity to elements.
