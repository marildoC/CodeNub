Understanding HTML Tables

Welcome to the section on HTML tables! In this part of the course, you'll learn about creating structured layouts using tables.

The Basics: Creating a Simple Table

HTML tables are used to organize data into rows and columns. Here's how you can create a basic table:

    <table>
      <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
      </tr>
      <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
      </tr>
    </table>

Adding Headers and Footers

You can enhance your tables by adding header and footer sections:

    <table>
      <thead>
        <tr>
          <th>Header 1</th>
          <th>Header 2</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Row 1, Cell 1</td>
          <td>Row 1, Cell 2</td>
        </tr>
        <tr>
          <td>Row 2, Cell 1</td>
          <td>Row 2, Cell 2</td>
        </tr>
      </tbody>
      <tfoot>
        <tr>
          <td colspan="2">Footer content</td>
        </tr>
      </tfoot>
    </table>

Spanning Rows and Columns

You can span rows and columns using the rowspan and colspan attributes:

    <table>
      <tr>
        <td rowspan="2">Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
      </tr>
      <tr>
        <td>Row 2, Cell 2</td>
      </tr>
      <tr>
        <td colspan="2">Row 3, Cell 1 and 2</td>
      </tr>
    </table>

Styling Tables with CSS

You can apply CSS styles to your tables to enhance their appearance. Here's an example:

    table {
      width: 100%;
      border-collapse: collapse;
    }
    
    th, td {
      border: 1px solid black;
      padding: 8px;
      text-align: left;
    }
