Understanding HTML Forms

Welcome to the section on HTML forms! In this part of the course, you'll learn about creating interactive forms to collect data from users.

The Basics: Creating a Simple Form

HTML forms are used to collect user input. Here's how you can create a basic form:

    <form>
      <label for="username">Username:</label>
      <input type="text" id="username" name="username"><br>
      
      <label for="password">Password:</label>
      <input type="password" id="password" name="password"><br>
      
      <input type="submit" value="Submit">
    </form>

Form Elements

HTML provides various form elements for different types of input:

  • <input type="text">: Text input field
  • <input type="password">: Password input field
  • <input type="checkbox">: Checkbox
  • <input type="radio">: Radio button
  • <input type="submit">: Submit button
  • <input type="reset">: Reset button
  • <select>: Dropdown list
  • <textarea>: Text area

Form Attributes

Forms can have attributes like action, method, and enctype:

    <form action="/submit-form" method="post" enctype="multipart/form-data">
      
    </form>

Styling Forms with CSS

You can apply CSS styles to your forms to enhance their appearance:

    form {
      width: 300px;
      margin: 0 auto;
    }
    
    input[type="text"], input[type="password"] {
      width: 100%;
      padding: 10px;
      margin-bottom: 10px;
    }
