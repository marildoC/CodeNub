Python File Handling

Welcome to the Python File Handling course! In this module, we'll explore how to work with files in Python, including reading from and writing to files.

Introduction to File Handling

File handling is an essential aspect of programming, allowing us to interact with files on our computer's file system. Python provides built-in functions and methods to perform various file operations.

Opening and Closing Files

Before we can read from or write to a file, we need to open it. Python's open() function is used to open a file. It takes two parameters: the file path and the mode in which to open the file (read mode, write mode, append mode, etc.).

# Open a file in read mode
file = open("example.txt", "r")

# Close the file
file.close()

Reading from Files

Python offers various methods for reading from files. The read() method reads the entire contents of the file, while readline() reads a single line at a time. Additionally, we can use a loop to iterate over each line in the file.

# Read the entire file
content = file.read()

# Read a single line
line = file.readline()

# Iterate over each line
for line in file:
    print(line)

Writing to Files

To write to a file, we open it in write mode ("w"). We can then use methods like write() or writelines() to write data to the file. Be careful, as opening a file in write mode will overwrite its existing contents.

# Open a file in write mode
file = open("example.txt", "w")

# Write data to the file
file.write("Hello, world!\n")
file.write("Python is awesome!\n")

# Close the file
file.close()

Working with File Paths

Python's os module provides functions for working with file paths and directories. We can use os.path.join() to construct file paths and os.path.exists() to check if a file exists.

import os

# Construct file path
file_path = os.path.join("folder", "example.txt")

# Check if file exists
if os.path.exists(file_path):
    print("File exists")
else:
    print("File does not exist")

Exception Handling

When working with files, it's essential to handle exceptions properly to avoid program crashes and data loss. Python's try and except blocks can be used for error handling.

try:
    file = open("example.txt", "r")
    content = file.read()
    file.close()
except FileNotFoundError:
    print("File not found")
except Exception as e:
    print("An error occurred:", e)

File Handling Best Practices

As with any programming task, there are best practices to follow when working with files. These include properly closing files after use, using context managers (with statement) to automatically close files, and handling exceptions gracefully.
