Mastering Collections in Java

Welcome to our comprehensive guide on collections in Java! In this tutorial, we'll explore the vast world of collections, indispensable tools for managing and manipulating groups of objects in Java.

Introduction to Collections

In Java, a collection is an object that represents a group of elements. Collections provide a flexible and efficient way to store, retrieve, and manipulate data. They offer various operations such as adding, removing, searching, sorting, and iterating over elements.

Types of Collections

Java offers a rich set of interfaces and classes for working with collections. Some of the most commonly used collection interfaces include:

  • List: An ordered collection that allows duplicate elements. Implementations include ArrayList, LinkedList, and Vector.
  • Set: A collection that does not allow duplicate elements. Implementations include HashSet, TreeSet, and LinkedHashSet.
  • Map: An object that maps keys to values. Implementations include HashMap, TreeMap, and LinkedHashMap.
  • Queue: A collection used for holding elements prior to processing. Implementations include PriorityQueue and LinkedList.

Using Collections in Java

Let's explore some common operations performed on collections:

Creating a Collection

To create a collection, you can instantiate one of the collection classes provided by Java. For example:

List<String> names = new ArrayList<>();

Adding Elements

You can add elements to a collection using the add method:

names.add("Alice");
    names.add("Bob");
    names.add("Charlie");

Iterating Through a Collection

You can iterate through the elements of a collection using loops or iterators:

for (String name : names) {
        System.out.println(name);
    }

Let's practise your understanding about Collections

Java code that demonstrates the declaration and initialization of Collections 
import java.util.*;
public class CollectionsExample {
    public static void main(String[] args) {
        // Creating an ArrayList to store integers
        List numbers = new ArrayList<>();

        // Adding elements to the ArrayList
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);

        // Creating a LinkedList to store strings
        LinkedList names = new LinkedList<>();

        // Adding elements to the LinkedList
        names.add("Alice");
        names.add("Bob");
        names.add("Charlie");

        // Creating a HashSet to store floating-point numbers
        Set prices = new HashSet<>();

        // Adding elements to the HashSet
        prices.add(10.5);
        prices.add(20.75);
        prices.add(30.0);

        // Creating a HashMap to store student names and their ages
        Map studentAges = new HashMap<>();

        // Adding elements to the HashMap
        studentAges.put("Alice", 25);
        studentAges.put("Bob", 30);
        studentAges.put("Charlie", 28);

        // Iterating through the ArrayList and printing elements
        System.out.println("Numbers:");
        for (int num : numbers) {
            System.out.println(num);
        }

        // Iterating through the LinkedList and printing elements
        System.out.println("\nNames:");
        for (String name : names) {
            System.out.println(name);
        }

        // Iterating through the HashSet and printing elements
        System.out.println("\nPrices:");
        for (double price : prices) {
            System.out.println(price);
        }

        // Iterating through the HashMap and printing key-value pairs
        System.out.println("\nStudent Ages:");
        for (Map.Entry entry : studentAges.entrySet()) {
            System.out.println(entry.getKey() + ": " + entry.getValue());
        }
    }
}
